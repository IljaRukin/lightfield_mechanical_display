# lightfield_mechanical_display
mechanics and rendering for a light field display to show images at different depth planes.<br>
The mechanical motor stage was a proof of concept for moving the lens array different distances from the lcd display and therefore selecting a particular depth plane for the observed image. In this configuration it's not accurate enough to be useful.
